# btree_py
Btree+ in python

btree does not mean binaria tree but named after Rudolf Bayer

## how to run

```
pythno btree.py
```

## how to change of btree_plus

```
	1.	キー: 1, 値: “one”
	2.	キー: 4, 値: “two”
	3.	キー: 6, 値: “three”
	4.	キー: 3, 値: “four”
	5.	キー: 7, 値: “five”
	6.	キー: 2, 値: “six”
	7.	キー: 5, 値: “seven”
```

### 1. キー: 1, 値: “one”
```
[1]
```

### 2. キー: 4, 値: “two”
```
[1, 4]
```

### 3.	キー: 6, 値: “three”
```
[1, 4, 6]
```

    [4]
   /   \
[1]    [4, 6]
```

### 4.	キー: 3, 値: “four”
```
     [4]
    /   \
[1, 3]  [4, 6]
```

### 5.	キー: 7, 値: “five”
```
     [4]
    /   \
